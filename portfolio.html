Toned Tribe Project

My first project for my GT-FT Bootcamp! My team created a webpage with the purpose of function as a geolocator and motivational quotes generator. The concept of the webpage was designed to give users the ease of locating nearby gyms and markets in their desired city. Along with that, we provided the users a generated motivational quote to encourage a fitness-related mindset.

One of the homework I had alot of fun doing. This homework tasked us to create a random password for users. The user has the ability to set the parameters of what their password consists of. The generator will follow those parameters and give a different random password each time.

Day Planner

For this homework I was assigned to create a Weekly Day Planner. This Day planner displayed the current date and time for the user. It also allow for user input where they can store their weekly plans. Whatever the user inputs into the planner will be saved into local storage and contine to stay there even if the user refreshes the page.
